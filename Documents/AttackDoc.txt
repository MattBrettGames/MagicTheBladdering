//== how to setup an attack ==\\

1. anim
2. animation tree
3. stick Weapon (or child) script on relevants weapon(s)
4. jam characterEventHandler on the gameObject that has the animator on it
5. set up the BeginAttack & EndAttack anim events
6. set up the BeginActing & EndActing anim events
7. add other anim events (such as Songbird's VialThrow())
8. apply other bonuses, such as hyper armour
